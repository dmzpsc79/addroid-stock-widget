1. 수집하는 개인정보

주식 위젯(이하 "앱")은 어떠한 개인정보도 수집하지 않습니다.

  • 회원가입·로그인 기능 없음
  • 이름, 이메일, 전화번호 등 식별 정보 수집 없음
  • 위치 정보 수집 없음
  • 광고 식별자(GAID) 수집 없음

2. 앱이 접근하는 데이터

앱은 주가 정보를 표시하기 위해 다음 외부 서비스에 네트워크 요청을 보냅니다.

  • 네이버 금융 (polling.finance.naver.com) — 국내 주가 조회
  • Yahoo Finance (query1.finance.yahoo.com) — 네이버 금융 조회 실패 시 대체 조회

이 요청은 종목 코드만 포함하며, 사용자를 식별하는 정보는 전송되지 않습니다.

3. 기기 내 저장 데이터

앱은 다음 정보를 기기 내 SharedPreferences에만 저장합니다.

  • 사용자가 설정한 종목 목록 (종목 코드·종목명)
  • 위젯 갱신 주기 설정
  • 마지막으로 조회한 주가 캐시

이 데이터는 기기 외부로 전송되지 않으며, 앱 삭제 시 자동으로 삭제됩니다.

4. 제3자 제공

앱은 어떠한 정보도 제3자에게 제공·판매·공유하지 않습니다.

5. 어린이 개인정보 보호

앱은 만 14세 미만 아동을 대상으로 하지 않으며, 해당 연령대의 개인정보를 의도적으로 수집하지 않습니다.

방침이 변경될 경우 이 페이지를 통해 공지합니다. 중요한 변경 사항은 앱 업데이트 노트에도 안내합니다.
